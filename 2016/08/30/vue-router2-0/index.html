vue-router 2.0

发表于 2016-08-30   |     |  

前言

最近在学习vue-router，发现2.0版本还是有些变化的，一些基本语法也做了些调整，毕竟2.0正式版本还没有发布，所以我就简单整理了一些常用语法的变化。

router-link

v-link改为router-link标签

下面是两种写法：

routes

router.map集成到routes中（包括redirect等）

mode有三个选项：history，abstract，hash（默认）

base就是原来的root

$mount()

之前的版本中用router.start来启动路由，2.0中用$mount()来发布到父组件中

children

在2.0中用children来代替subRoutes，写法也有细微变化

其他的一些改变

router.push取代router.go

beforeEnter取代canActivate

beforeRouteLeave取代canDeactivate

canReuse被移除

等等……

有需要的同学可以去看一下 2.0的RC文档，也可以对照着 0.7~版本文档 看一看。
